MainApp (tabbed interface)\n  └─ If not → AuthStack (LoginScreen/SignUpScreen)\n\nAuthContext (useAuth hook)\n  ├─ Wraps Redux dispatch\n  ├─ Manages AsyncStorage persistence  \n  └─ Exposes simple API: login(), signUp(), logout()\n\nAuthSlice (Redux)\n  ├─ loginAsync thunk (awaits 1s, validates, returns user+token)\n  ├─ signUpAsync thunk (same)\n  └─ Manages loading, error, success states\n\nComponents\n  ├─ Local form state (useReducer)\n  ├─ Client validation\n  └─ Display auth state (loading, errors, success)\n```\n\n## Quick Test\n\n1. **Start App**\n   ```bash\n   npm
